

მთავარ გვერდზე შეიძლება თამაშების დამატება, წაშლა, ძებნით ფილტრაცია, თამაშების საკუთარ ბიბლიოთეკაში დამატება.
თამაშები ემატება რანდომული სურათით ([picsum](https://picsum.photos/)) და რეიტინგით.

მეორე გვეერდზე საკუთარი ბიბლიოთეკის ნახვა და მხოლოდ საკუთარი ბიბლიოთეკიდან წაშლაა შესაძლებელი.

მომხმარებელს საბოლოო აპლიკაციაში თამაშების დამატება და წაშლა არ შეეძლება(მხოლოდ ბიბლიოთეკაში)
უბრალოდ შუალედურის პირობისთვისაა.

ფინალურისთვის თამაშების ჩატვირთვა და ფილტრაციაც api-ით მოხდება,
firebase-ით ავტენთიფიკაცია/ავტორიზაცია, ბიბლიოთეკების შენახვა ბაზაში,
მეტი ვიზუალური კომპონენტების და დამატებითი ფუნქციონალის დამატებასაც ვაპირებ.
